# rmv

Remove similar tickers to compare with.

### Usage

```python
rmv [-s L_SIMILAR]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| l_similar | Tickers to remove from similar list |  | True | None |
